# Commands


## 0x18  Get current background color
Get the background color by index

### Parameter
| Register        | Decription                                |
| -               | -                                         |
| DIS00           | The palette index of the current bgcolor. |


## 0x19  Set current background color
Set the background color by index

### Return
| Register        | Decription                                |
| -               | -                                         |
| DIS00           | The palette index of the current bgcolor. |
